Developed a production-ready Python desktop utility that automates data extraction from SQL databases into structured Excel sheets.

Key Achievements & Engineering Deliverables:
- Automated ETL Pipeline: Engineered efficient data read connections via Pandas for automated relational querying.
- Dynamic Paths & Serialization: Configured robust OS path logic and automated time-stamped serialization.
- Interactive Desktop GUI: Built an intuitive desktop interface using Tkinter with multi-language dialog confirmations.

Empowers non-technical users to run data pulls in one click.
